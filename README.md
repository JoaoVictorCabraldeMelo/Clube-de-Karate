## Getting Started

Este e um exemplo de como buildar o código

### Installation

1. Clone the repo
```sh
git clone https:://github.com/JoaoVictorCabraldeMelo/Clube-de-Karate
```
2. Change the directory 
```sh
cd Clube-de-Karate
```
3. Execute in the terminal
```sh
g++ clubekarate.cpp
```
4. Execute the a.out
```sh
./a.out
```
